=== cpu prop ===
Name — Имя процессора (например, "Intel(R) Core(TM) i7-9700K CPU @ 3.60GHz").
Manufacturer — Производитель процессора (например, "GenuineIntel" или "AuthenticAMD").
ProcessorId — Уникальный идентификатор процессора.
NumberOfCores — Количество физических ядер.
NumberOfLogicalProcessors — Количество логических процессоров (потоков).
MaxClockSpeed — Максимальная тактовая частота процессора (в МГц).
CurrentClockSpeed — Текущая тактовая частота (в МГц).
L2CacheSize — Размер кэша L2 (в КБ).
L3CacheSize — Размер кэша L3 (в КБ).
Architecture — Архитектура процессора (например, 0 для x86, 9 для x64).
AddressWidth — Разрядность адресов, поддерживаемых процессором (например, 32 или 64 бита).
Caption — Краткое описание процессора.
Description — Подробное описание процессора.
DeviceID — Уникальный идентификатор устройства процессора.
Family — Семейство процессоров (например, Intel Core i7).
SocketDesignation — Тип и название сокета, к которому подключен процессор.
LoadPercentage — Процент текущей нагрузки на процессор.
